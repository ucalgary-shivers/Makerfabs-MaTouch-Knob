# Arduino libmapper – "Example_ESP32.ino" Running Guide

This sketch is an example from the [libmapper-arduino](https://github.com/libmapper/libmapper-arduino) repository and is used for testing within this project.

## Steps

1. Update the Wi‑Fi login parameters:

   ```cpp
   const char* ssid = "WIFI_SSID";
   const char* password = "WIFI_PASSWORD";
    ```
2. Upload the sketch to the MaTouch Knob using the same process as uploading the firmware. For detailed instructions, please refer to the Read.md file in the root directory.
